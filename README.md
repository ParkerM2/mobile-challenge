* Node / NPM
	
## Setup
To run this project:

Run: 
```
git clone https://github.com/ParkerM2/mobile-challenge.git
```

To install node modules:
```
cd mobile-challenge/
npm install
```
To run, use: 
```
npm start
```
To test, use:
```
 npm test
```
