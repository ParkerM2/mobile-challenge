## Technologies
Project is created with:
* React Native
* API / Fetch
* Expo
* JavaScript
* CSS
* Jest
* Node / NPM
	
## Setup
To run this project:

```
Run: git clone https://github.com/ParkerM2/mobile-challenge.git
Install node modules: npm install
To run use: npm start or expo start
```
